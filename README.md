## 🌈✨ About The Project

**FoodExpress** is a front-end web project that simulates a real online food delivery experience.  
Users can explore restaurants, filter categories, add food items to a cart, and place demo orders — all in a smooth and interactive UI.

💡 Perfect for learning **DOM manipulation**, **UI/UX design**, and **JavaScript logic**.

---

## 🌟✨ Key Features

🍔 Beautiful & responsive UI  
🧭 Sticky navigation bar  
🔍 Search food by delivery location  
📂 Food categories with icons  
🏪 Dynamic restaurant listing  
🎯 Filter restaurants (Top Rated • Fast • Offers)  
🛒 Add to cart with quantity control  
💰 Automatic price calculation  
🎉 Animated notifications  
🧾 Checkout simulation  

---

## 🎨✨ Tech Stack

🧱 **HTML5** – Page structure  
🎨 **CSS3** – Styling, animations & layout  
⚙️ **JavaScript (Vanilla JS)** – Dynamic content & cart logic  
📐 **Flexbox & Grid** – Responsive layouts

# 🧠✨ How It Works

🟢 Restaurants are loaded dynamically using JavaScript
🟢 Filters update restaurant cards instantly
🟢 Clicking a restaurant adds demo items to cart
🟢 Cart auto-updates quantities & total price
🟢 Checkout clears the cart and confirms the order
